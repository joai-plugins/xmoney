# xMoney

Connect xMoney to Claude, Codex, and ChatGPT through JoAi's hosted MCP app server.

[Open app page](https://joai.ai/en/apps/xmoney) · [Browse all apps](https://joai.ai/apps)

## Works With

- ChatGPT
- Claude
- Codex
- Cursor
- Other MCP clients

## Install

### Claude

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-xmoney@joai-claude-plugins
```

### Codex

```bash
codex mcp add joai-xmoney --url https://cortex.joai.ai/mcp/apps/xmoney
```

### Cursor

Copy to local plugins:

```bash
git clone https://github.com/joai-plugins/xmoney.git ~/.cursor/plugins/local/joai-xmoney
```

Or use the [MCP deeplink](cursor://anysphere.cursor-deeplink/mcp/install?name=joai-xmoney&config=eyJtY3BTZXJ2ZXJzIjp7ImpvYWkteG1vbmV5Ijp7InVybCI6Imh0dHBzOi8vY29ydGV4LmpvYWkuYWkvbWNwL2FwcHMveG1vbmV5In19fQ==) to install directly.

### ChatGPT

Paste the MCP URL in ChatGPT settings:

```
https://cortex.joai.ai/mcp/apps/xmoney
```
